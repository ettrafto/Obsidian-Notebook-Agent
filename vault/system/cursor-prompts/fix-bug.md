# Fix Bug

## Goal
Fix a bug with a clear reproduction and expected behavior.

## Context
- Provide reproduction steps
- Specify expected vs actual behavior

## Constraints
- Changes must be minimal and scoped to the bug
- No contract changes unless ADR exists

## Files in Scope
- List affected files

## Acceptance Criteria
- Bug is fixed per reproduction steps
- Verification steps recorded
- No regressions introduced

## Rollback Plan
- Revert the fix commit if verification fails
